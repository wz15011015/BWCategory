## Introduction

项目中常用的一些分类整理，如下：

```
1. UIDevice+HDLHelper
2. NSBundle+HDLHelper
3. NSString+HDLHelper
4. UIImage+HDLHelper
5. UIColor+HDLHelper
6. UIAlertController+HDLHelper
7. NSArray+HDLHelper
```


## Usage

###### 1. 导入 HDLHelper 文件夹到项目中
###### 2. 在需要使用的类中导入头文件: ```#import "HDLHelper.h"```
